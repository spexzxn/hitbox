# Hitbox Mod для Minecraft 1.16.5

Мод для отображения хитбоксов сущностей в Minecraft версии 1.16.5 на Forge.

## Возможности

- Отображение хитбоксов всех сущностей в настраиваемом радиусе
- Переключение отображения клавишей **H**
- Открытие настроек клавишей **K**
- Настраиваемые цвета для разных типов сущностей:
  - Настраиваемый цвет для живых сущностей (мобы, игроки) - по умолчанию синий
  - Настраиваемый цвет для неживых сущностей (предметы, стрелы и т.д.) - по умолчанию красный
- Настройка радиуса отображения (1-256 блоков)
- Опция показа хитбоксов по умолчанию при входе в мир

## Установка

1. Установите Minecraft Forge версии 36.2.39 для Minecraft 1.16.5
2. Соберите мод с помощью Gradle: `./gradlew build`
3. Поместите собранный jar файл в папку `mods` вашей игры

## Сборка

```bash
./gradlew build
```

Собранный мод будет находиться в `build/libs/`

## Использование

- **H** - переключить отображение хитбоксов
- **K** - открыть экран настроек

## Настройки

В экране настроек (клавиша **K**) вы можете изменить:

1. **Цвета для живых сущностей** (RGB значения от 0.0 до 1.0)
   - По умолчанию: R=0.0, G=1.0, B=1.0 (синий)

2. **Цвета для неживых сущностей** (RGB значения от 0.0 до 1.0)
   - По умолчанию: R=1.0, G=0.0, B=0.0 (красный)

3. **Радиус отображения** (от 1 до 256 блоков)
   - По умолчанию: 64 блока

4. **Показывать по умолчанию** - включить/выключить автоматический показ хитбоксов при входе в мир

Конфигурационный файл также сохраняется в `config/hitboxmod-client.toml` и может быть отредактирован вручную.

## Требования

- Minecraft 1.16.5
- Minecraft Forge 36.2.39 или выше
- Java 8
